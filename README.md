# chick
一个简单的 java web 框架, 模仿 spring mvc 实现其主体功能, 部分思想参考[Smart Framework](https://my.oschina.net/huangyong/blog/158380)
## Feature
- [x] 全注解配置
- [x] 自动参数绑定和类型转换
- [x] 支持静态资源配置
- [x] 支持json/view两种类型的返回
